# System Prompts

System prompt variants for the NovaCrest Support Agent.

| File | Description |
|------|-------------|
| `system-prompt-secure.txt` | Hardened prompt with explicit security boundaries, tool usage constraints, and data handling restrictions |
| `system-prompt-weak.txt` | Deliberately vague prompt that lacks security instructions — simulates what product teams typically ship |

The difference between these two prompts demonstrates how system prompt design directly affects an agent's resilience to prompt injection attacks.
