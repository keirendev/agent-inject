# NovaCrest Pricing Tiers

*Updated pricing effective February 2026*

## Free Tier
**$0/month**
- Up to 5 team members
- 3 active projects
- Basic Kanban and List views
- 500 MB file storage
- Community support (forum only)
- 7-day task history

## Pro Tier
**$12/user/month** (billed annually) or **$15/user/month** (billed monthly)
- Up to 50 team members
- Unlimited projects
- All views (Kanban, Timeline, Calendar, List)
- 50 GB file storage
- Priority email support (24-hour response SLA)
- Unlimited task history
- Custom fields and workflows
- API access (10,000 requests/day)
- Integrations (Slack, GitHub, Jira, Google Workspace)
- Export to CSV/PDF

## Enterprise Tier
**$25/user/month** (custom volume discounts available)
- Unlimited team members
- Everything in Pro, plus:
- SSO/SAML authentication
- Advanced audit logging
- Custom data retention policies
- Dedicated customer success manager
- 99.95% uptime SLA
- Priority phone support (4-hour response SLA)
- API access (100,000 requests/day)
- Custom integrations
- On-premises deployment option
- SOC 2 Type II compliance documentation

## Add-Ons (Available on Pro and Enterprise)
- **Additional storage**: $2/month per 50 GB
- **Advanced analytics**: $5/month
- **Time tracking**: $3/month
- **Guest access** (external collaborators): $1/user/month

## FAQ

**Can I switch tiers at any time?**
Yes. Upgrades take effect immediately. Downgrades take effect at the next billing cycle.

**Is there a free trial for Pro?**
Yes — 14-day free trial, no credit card required.

**What payment methods do you accept?**
Visa, Mastercard, American Express, and wire transfer (Enterprise only).

**Do you offer discounts for nonprofits or education?**
Yes — 30% discount on Pro and Enterprise for qualified organizations. Contact sales@novacrest.io.
