# NovaCrest Feature Overview

## Project Management

### Task Management
- Create, assign, and track tasks with custom statuses
- Set priorities (Critical, High, Medium, Low)
- Add due dates, time estimates, and dependencies
- Attach files, images, and links to tasks
- Comment threads on each task for collaboration

### Multiple Views
- **Kanban Board**: Visual workflow with drag-and-drop
- **Timeline (Gantt Chart)**: Plan project schedules and dependencies
- **Calendar View**: See deadlines at a glance
- **List View**: Sortable, filterable task tables

### Workflows
- Customize task statuses per project
- Set up automated transitions (e.g., move to "Review" when all subtasks complete)
- Create templates for repeating project types

## Collaboration

### Real-Time Editing
- Multiple team members can edit project details simultaneously
- Live cursor presence — see who's viewing what
- Instant updates across all connected clients

### Comments and Mentions
- @mention team members in task comments
- Thread replies for organized discussions
- Email notifications for mentions and updates

### File Sharing
- Drag-and-drop file uploads
- Preview images, PDFs, and documents in-app
- Version history for uploaded files

## Integrations (Pro and Enterprise)

### Communication
- **Slack**: Get task notifications in channels, create tasks from messages
- **Microsoft Teams**: Similar notification and creation features

### Development
- **GitHub**: Link pull requests to tasks, auto-update task status
- **GitLab**: Bi-directional sync for issues and merge requests
- **Jira**: Import projects and sync task status

### Productivity
- **Google Workspace**: Attach Drive files, sync Calendar
- **Zapier**: Connect to 5,000+ apps with no-code automations

## API
- RESTful API with full CRUD operations
- Webhook support for real-time events
- OAuth 2.0 authentication
- Rate limits: 10,000 req/day (Pro), 100,000 req/day (Enterprise)
- SDKs available for Python, JavaScript, and Go

## Security
- AES-256 encryption at rest
- TLS 1.3 for data in transit
- SOC 2 Type II certified (Enterprise)
- GDPR compliant
- Two-factor authentication
- Role-based access control (Admin, Manager, Member, Guest)
