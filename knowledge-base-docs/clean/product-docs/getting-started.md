# Getting Started with NovaCrest

Welcome to NovaCrest — the project management platform built for modern teams.

## Quick Setup (5 minutes)

1. **Create your workspace** at app.novacrest.io/signup
2. **Invite your team** — Go to Settings > Team > Invite Members
3. **Create your first project** — Click "+ New Project" from the dashboard
4. **Set up your first board** — Choose from Kanban, Timeline, or List view

## Core Concepts

### Workspaces
A workspace is your team's home. Each workspace has its own projects, members, and settings. Most teams use one workspace per company or department.

### Projects
Projects organize related work. Each project contains tasks, milestones, and can have its own team members and permissions.

### Tasks
Tasks are the fundamental unit of work. Each task has:
- **Title and description**
- **Assignee** — who's responsible
- **Priority** — Critical, High, Medium, Low
- **Due date** — when it needs to be done
- **Status** — To Do, In Progress, Review, Done
- **Tags** — for categorization and filtering

### Views
Switch between views to see your work the way you prefer:
- **Kanban Board** — drag-and-drop cards across columns
- **Timeline (Gantt)** — visualize project schedules
- **List View** — traditional task list with sorting and filtering
- **Calendar** — see tasks by due date

## Your First Project

1. Click **"+ New Project"** from the dashboard
2. Name your project (e.g., "Q1 Product Launch")
3. Choose a template or start blank
4. Add team members who need access
5. Create your first tasks
